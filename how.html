Strategy 1:

  • Find a scenario where the same person says "it was the" or "it wasn't the" twice.
  • If that scenario is found put the 2 cards referrenced into a suspects list
  • Put the remaining cards into the innocent list
  • If a card is in the innocent list we are able to determine if an accusation is now a Truth/Lie.
  • If a card is in the suspect list we mark those accusations as Not Sure.
  • Since each person tells 1 truth and 1 lie we are able to fillin
  • Every accusation we've set to truth or lie we've stored that solution as well as the solution to the contradicting accusation.
  • We then go through all the accusations and check if a solution exists for each one.
  • Finally the table should be mostly filled out it will go through and check for an accusation being set to "Lie"" and having the prefix of "It wasn't the" or an accusation being set to "Truth"" and the prefix "it was the".
  • If this doesn't find a card it moves on to strategy 2

Strategy 2:

  • If there are cards in the suspect list flip a coin and guess one of those.
  • If there are not cards in suspect list display that there is not enough data to make a prediction.
